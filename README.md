# Combinators

## 1. 개요
어떤 것을 선택해야 할 지 고민이 많은 사람들을 위해 대신 선택해주는 자신만의 선택 도우미를 목표로
만들었습니다.

## 2. 기술스택
Flutter, Dart, Bloc, Sqlite, Firebase Analytics, Admob

## 3.
1. Sqlite에 카테고리와 아이템들을 저장 하도록 하였습니다.
2. 1시간에 30번 무작위 조합을 다시 얻을 수 있습니다. 횟수를 소모 하였다면 광고로 다시 충전 가능합니다.
